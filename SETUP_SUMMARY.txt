UNIQUE FEATURES VS. STANDARD PACKAGES
================================================================================

Standard 3SLS packages (e.g., systemfit):
   - Basic 3SLS estimation only
   - Limited weight handling
   - No equation-specific subsets
   - Minimal output options

This implementation adds:
   ✓ Full survey weight support throughout
   ✓ Equation-specific sample restrictions via | subset()
   ✓ Advanced LaTeX output functions
   ✓ Robust variance-covariance estimates
   ✓ Automatic factor analysis utilities
   ✓ Polychoric transformation for ordinal data
   ✓ Reduced form with impact multipliers

Perfect for:
   - Survey data analysis
   - Heterogeneous sample restrictions
   - Publication-ready output
   - Teaching & methodology
